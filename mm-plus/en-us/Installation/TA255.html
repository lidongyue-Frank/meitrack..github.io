TA255 series product installation and commissioning

 

If you are using it for the first time, please read this installation and commissioning guide carefully.

Step 1.  Prepare for installation and commissioning

1. The tool

A.L type wrench (please choose standard or the same specifications)

B. Magnetic charging cable (please use standard products)

2. The software

A.Meitrack Manager Plus(IOS or Android)

Note: Please download as required

3. The SIM card

A.NANO SIM card

Step 2. Turn on the device

A. Use an L-shaped wrench or A tool of the same specifications to open the host cover

Step 3. Install the card

A. Locate the position of the card slot and install the card in the direction indicated

Note: Make sure that the SIM card is open and has certain traffic

Step 4. Debug parameters

A. Make sure the battery is loaded and turn on the power switch (Bluetooth function will be enabled at this time and will last for 5 minutes)

Note: There are three ways to turn on Bluetooth:

1. Switch on and off the machine (Bluetooth on for 5 minutes)

2. Connect the magnetic charger to the power supply for charging (Bluetooth on for 5 minutes)

3. Connect to battery box (Bluetooth on for 5 minutes)

B. Ensure that Meitrack Manager Plus is installed successfully and Bluetooth is enabled

C. Open Bluetooth Connection on the home page of Meitrack Manager Plus and search for the device you want to find

D. Make a Bluetooth connection

E. Locate the Parameter Settings menu and set related parameters

Note: To ensure that the device can go online, ensure that the domain name, IP address, port number, and APN information are correct

Step 5. Query the status

A. After the Bluetooth connection, find Device Status to view the device status

B. Ensure that the device status, device signal, location, and battery voltage are normal

C. Ensure that the status indicator is Online. Otherwise, check the parameter Settings again

Step 6. Install the device cover

A. Cover the case and screw the case in place using an L wrench or the same tool

Note: Ensure that no impurities are inserted into the waterproof sealing ring and it is placed correctly

Step 7. Install devices

A. Select an installation mode as required

Note: The equipment can be installed by means of high intensity magnetic, cable tie, 3M glue, glass glue, etc. If high intensity magnetic installation is required, please use high intensity magnetic tablets to fix them at the screw holes of the equipment.
